# Visual Mappings: Relational Wholes (Rn) & Projections (Pk) Flow Diagrams

## Overview

This document provides detailed visual mappings and diagrams showing how **Relational Wholes (Rn)** and **Projections (Pk)** flow between active interfaces in the cosmos system architecture.

## 1. System-5-CNS-ORG Diagram Analysis

### 1.1 Diagram Structure

The **System-5-CNS-ORG.png** diagram shows the **Company & Nervous System Integration** with three main triads arranged in a tetrahedral structure:

```
                    Neocortex (Yellow - Cerebral Triad)
                           /  \
                          /    \
                         /      \
                        /        \
                       /          \
          Basal System            Limbic System
       (Light Blue - Somatic)  (Turquoise - Autonomic)
```

### 1.2 Service Distribution

**Cerebral Triad (Yellow) - Top**:
- **T-7** (Pink circle): Thought Service - Right Hemisphere (Intuitive Ideas, Potential)
- **PD-2** (Pink circle): Processing Director - Central Coordination
- **P-5** (Blue circle): Processing Service - Analytical Processing
- **O-4** (Blue circle): Output Service - Left Hemisphere (Applied Techniques, Commitment)

**Somatic Triad (Light Blue) - Bottom Left**:
- **M-1** (Green circle): Motor Control - Basal System (Somatic Balance, Performance)
- **S-8** (Green circle): Sensory Service - Environmental Input
- **P-5** (Blue circle): Processing Service - Behavioral Techniques
- **O-4** (Blue circle): Output Service - Motor Response

**Autonomic Triad (Turquoise) - Bottom Right**:
- **M-1** (Green circle): Monitoring Service - System Health
- **S-8** (Green circle): State Management - Limbic System (Emotive Balance, Performance)
- **PD-2** (Pink circle): Process Director - Background Orchestration
- **P-5** (Blue circle): Processing Service - Emotive Processing
- **T-7** (Pink circle): Trigger Service - Autonomic Responses (Parasympathetic Polarity, Intuitive Feeling, Potential)

### 1.3 Polarity Color Coding

**Pink Circles (Sympathetic Polarity)**:
- T-7 services (Thought, Trigger)
- Represents **active responses, alertness, immediate reactions**
- Maps to **Potential Dimension [D-T] (2-7)**

**Blue Circles (Somatic Polarity)**:
- P-5 and O-4 services (Processing, Output)
- Represents **skill execution, behavioral patterns, voluntary motor control**
- Maps to **Commitment Dimension [P-O] (5-4)**

**Green Circles (Performance Polarity)**:
- M-1 and S-8 services (Motor/Monitoring, Sensory/State)
- Represents **external interface, state management, performance optimization**
- Maps to **Performance Dimension [S-M] (8-1)**

**Pink Circles (Parasympathetic Polarity)**:
- PD-2 services (Processing Director)
- Represents **rest, maintenance, optimization, homeostasis**
- Shared between Somatic and Autonomic triads
- Maps to **Potential Dimension [D-T] (2-7)**

## 2. Relational Wholes (Rn) Flow Diagrams

### 2.1 System 2: Basic Countercurrent Pattern

```
┌─────────────────────────────────────────────────────┐
│                   System 2                          │
│                                                     │
│        C1 (Idea/Electronic)                        │
│              ↓ R1                                   │
│              ↓ (Subjective direction)              │
│        C2 (Knowledge/Cell) ← PIVOT                 │
│              ↓ R1                                   │
│              ↓                                      │
│        C3 (Routine/Body)                           │
│              ↑                                      │
│              ↑ R2                                   │
│              ↑ (Objective feedback)                │
│        C2 (Knowledge/Cell) ← PIVOT                 │
│              ↑ R2                                   │
│              ↑                                      │
│        C1 (Idea/Electronic)                        │
│                                                     │
│  Countercurrent Balance: R1 ↔ R2 across C2 pivot  │
└─────────────────────────────────────────────────────┘
```

**Key Properties**:
- **R1**: Idea (C1) directs routines (C2) to give form (C3)
- **R2**: Form (C3) feeds back through sensory routines (C2) to idea (C1)
- **Pivot**: C2 acts as perceptual axis balancing subjective and objective
- **Closed Circuit**: Energy efflux from L0 in C1 returns to L0 in C1
